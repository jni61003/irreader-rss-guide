## irreader 自定义订阅功能：怎么订阅没有 RSS 的网站

这是 irreader 最值得单独讲的功能。操作流程如下：

1. 打开 irreader，点击左上角「添加订阅源」
2. 在地址栏粘贴目标网页的链接，点击加载
3. 软件会显示网页内容，这时候右键点击你想订阅的内容标题（比如文章标题、视频标题）
4. 软件自动计算订阅规则，识别出这类内容的更新模式
5. 点击「开始订阅」，完成

之后这个页面有新内容，就会自动推送到你的阅读列表。

从实际使用来看，绝大多数内容类网站都能成功订阅。少数结构比较特殊的网站可能需要手动微调规则，但官方也有配套的视频教程，上手不难。
